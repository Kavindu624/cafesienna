# ☕ CafeSienna – Coffee Shop Website

## 📌 Project Overview

CafeSienna is a modern and responsive **coffee shop website** designed to showcase a café’s brand, menu, and customer experience in an elegant digital format.

The website focuses on creating a warm and inviting online presence for the café while providing customers with easy access to menu details, location, and contact information.

---

## 🎯 Objectives

- Create a visually appealing coffee shop website  
- Showcase menu items in an organized and attractive way  
- Provide essential café information (location, contact, hours)  
- Improve customer engagement through a clean UI/UX design  
- Build a fully responsive website for all devices  

---

## 🌐 Key Features

- 🏠 Elegant landing page with café branding  
- ☕ Menu section displaying coffee, drinks, and food items  
- 📍 Location and contact information section  
- 🕒 Opening hours display  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🎨 Modern UI with smooth layout and styling  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript 
- Responsive Web Design (Flexbox / Grid)  

---

## 💡 Key Outcome

This project demonstrates how a **simple business website can be transformed into a professional digital presence**, improving customer accessibility and enhancing brand identity for a coffee shop.
